Cloud Kit Crypto

Required Module: cloudkit-crypto

PIN

Allows your app to store/read token protected with pin and encryption key in keystore

Init

String prefName = "tkd";
String keyMasterName = "km1"
...
PinTokenProvider pinTokenProvider = new PinTokenProviderImpl(this, prefName, keyMasterName);

Store a token

String tokenId = "tokenId";
String pin= getPin(); // could be alphanumeric
PinToken pinToken = pinTokenProvider.generateAndStore(tokenId, pin);
String tokenValue = pinToken.getValue();

// if you want to store your custom secret, use
String mySecret = "..."
PinToken pinToken = pinTokenProvider.store(tokenId, mySecret, pin);

Read token

String pin= getUserPin();
PinToken pinToken = pinTokenProvider.get(tokenId,pin);
String tokenValue = pinToken.getValue();

Delete token

PinToken pinToken = pinTokenProvider.delete(tokenId);

Biometric

Allows your app to store/read token protected with biometric

Initialize

BiometricSupport biometricSupport = BiometricSupport.of(this);

In the initiliaztion, biometric support performe a warnup in order to check the underling key material status. If key material is compromised, all data will be deleted an material reinit. You can performe a check programmatically using isBiometricKeyValid() api or BiometricHelper.isBiometricSupportValid(biometricSupport)

Check biometric status

BiometricStatus status = biometricSupport.getBiometricStatus();

switch (status){
    case ENROLLED:
        // A valid biometry is enrolled on this device
        break;
    case AVAILABLE_NOT_ENROLLED:
        // Device support biometric authentication but no biometry has currently enrolled
        break;
    case NOT_AVAILABLE:
        // No hardware available
        break;
    case ERROR:
        // Device signal an error
        break;
}

Store biometric token

this.authenticationPrompt = BiometricHelper.preparePrompt(this, biometricSupport, VCKPlatformExecutors.MAIN_THREAD_EXECUTOR);
authenticationPrompt.storeToken(tokenId, "MyTestFingerprintToken", new BiometricPrompt.PromptInfo.Builder()
        .setTitle("Your title")
        .setSubtitle("Your message")
        .setNegativeButtonText("cancel")
        .build(), new AuthenticationPrompt.Callback() {
    @Override
    public void onSuccess(BiometricToken biometricToken) {
        //do something with this token
    }


    @Override
    public void onAuthenticationError(VCKBiometricError biometricError) {
        super.onAuthenticationError(biometricError);
        Timber.e("onAuthenticationError -> [%d] : %s",biometricError.getOriginalCode(),biometricError.toString());
    }
});

Check if token is present

String tokenId = "myTokenId"

boolean biometricTokenAvailable = biometricSupport.isEncryptedDataAvailable(tokenId);

Read biometric token

// read token
this.authenticationPrompt = BiometricHelper.preparePrompt(this, biometricSupport, VCKPlatformExecutors.MAIN_THREAD_EXECUTOR);
    this.authenticationPrompt.getToken(tokenId, new BiometricPrompt.PromptInfo.Builder()
            .setTitle("Your title")
            .setSubtitle("Sub title")
            .setNegativeButtonText("Cancel")
            .build(), new AuthenticationPrompt.Callback() {
        @Override
        public void onSuccess(BiometricToken biometricToken) {
            //do something with this token
        }

        @Override
        public void onAuthenticationError(VCKBiometricError biometricError) {
            super.onAuthenticationError(biometricError);
            Timber.e("onAuthenticationError -> [%d] : %s",biometricError.getOriginalCode(),biometricError.toString());
        }
});

Delete biometric token

biometricSupport.deleteAllDataByKey(tokenId);

Re-init

biometricSupport.resetAndReInit()
